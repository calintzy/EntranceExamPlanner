## 2. 수집해야 할 데이터 항목

### A. 전형 기본정보
- 수시/정시 모집인원 및 비율
- 전형 유형별 선발인원 (교과, 종합, 논술, 실기, 수능)
- 전형 일정 (원서접수, 면접, 합격발표)

### B. 수능 관련
- 과목별 반영비율 (국어, 수학, 영어, 탐구)
- 수능 최저학력기준 (등급합, 필수 영역)
- 영어 절대평가 감점 기준
- 탐구 과목 조합 허용/제한 (과탐 I+II, 사탐+과탐 교차 등)
- 가산점 (수학 미적분/기하, 과탐 선택 시)

### C. 내신 관련
- 반영 교과 및 과목 (국영수과/전과목)
- 등급 산출 방식 (9등급제 vs 5등급제 전환)
- 학년별 반영비율
- 진로선택과목 반영 방법 (성취도 A/B/C)

### D. 생활기록부 (학종)
- 세부능력 및 특기사항(세특) 반영 여부 및 비중
- 창의적 체험활동(창체) - 정규 동아리만 반영
- 비교과 축소 현황 (수상, 독서, 자율동아리 미반영)
- 자기소개서 폐지 여부

### E. 교과이수 조건
- 필수 이수 과목 (서울대, 고려대, 연세대, 중앙대 등)
- 권장 과목 vs 필수 과목 구분
- 전공 연계 교과 이수 기준

### F. 대기업 연계학과
- 대학/학과/연계기업 매핑
- 모집인원 및 전형방법
- 혜택 (등록금, 취업보장, 인턴십)

## 4. 현실성 판단

### 가능한 것 (GREEN)
| 항목 | 근거 |
|------|------|
| 대학별 모집인원/전형 비율 | 교육부 PDF, adiga.kr에서 매년 공식 발표 |
| 수능 반영비율/최저기준 | 대학별 시행계획에 명시, 진학사 등에서 정리 |
| 대기업 연계학과 목록 | 13개교 18학과로 규모 작음, 뉴스/공식자료에서 수집 가능 |
| 교과이수 권장/필수과목 | 서울대, 고려대, 연세대, 중앙대 4개교가 공개 중 |
| 주요 일정 (D-day) | 대교협에서 통합 일정표 제공 |

### 노력 필요한 것 (YELLOW)
| 항목 | 과제 |
|------|------|
| 내신 반영방법 상세 | 대학별로 계산법이 다름 → 40+개 대학 개별 확인 필요 |
| 생활기록부 평가 기준 | 정성평가 영역이라 수치화 어려움, 대학별 가이드라인 수준 |
| 전형별 합격선/입결 | 대학이 공개하는 경우와 아닌 경우 혼재 |
| PDF 데이터 구조화 | 대학별 모집요강 PDF 파싱 필요 (표 형식 비표준) |

### 어려운 것 (RED)
| 항목 | 이유 |
|------|------|
| "입시 전략" 자동 생성 | 해석/판단이 필요한 영역, 법적 리스크 가능 |
| 실시간 경쟁률 데이터 | 원서접수 기간에만 존재, 접근 제한 |
| 학원가 비공개 노하우 | 유료 컨설팅 영역, 수집 불가 |

## 5. 결론 및 제안

### 현실성: 충분히 가능 (단, 범위 조절 필요)

**핵심 판단:**
- "팩트 데이터 구조화"는 현실적으로 가능
- "전략 자문"은 Phase 2 이후에 검토
- 데이터 소스는 충분히 존재하나, 자동 수집보다 **반자동 수집 + 수동 검증**이 현실적

### 추천 1단계 범위 (MVP)

**대상:** 인서울 TOP 15 대학 + 전국 의대 + 대기업 연계학과

**데이터 항목 (1단계):**
1. 대학별 수시/정시 모집인원 및 전형 유형
2. 수능 반영비율 및 최저학력기준
3. 내신 반영 교과 및 방법
4. 교과이수 필수/권장과목
5. 대기업 연계학과 매핑

**기술 접근:**
1. 교육부 PDF → 파싱하여 JSON/DB 구조화
2. 진학사/내신닷컴 공개 데이터 → 크로스체크용
3. 각 대학 입학처 모집요강 PDF → 핵심 항목 수동 추출 후 구조화
4. 연 1회 업데이트 (시행계획 발표 시점: 매년 4~5월)

### 추천 데이터 구조

```
대학별_입시정보/
├── universities/          # 대학 기본정보
├── admissions/            # 전형별 모집요강
│   ├── early/             # 수시
│   └── regular/           # 정시
├── subjects/              # 교과이수 조건
├── corporate-tracks/      # 대기업 연계학과
└── medical-schools/       # 의대 전용 정보
```
