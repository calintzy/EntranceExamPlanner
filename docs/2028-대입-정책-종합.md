# 2028학년도 대입 교과과정 & 입시정책 종합 가이드

## 1. 2022 개정 교육과정 — 선택과목 체계

2025년 고1부터 적용. 선택과목이 3단계로 재편:

| 구분 | 성격 | 예시 | 내신 평가 |
|------|------|------|-----------|
| **공통과목** | 모든 학생 필수 이수 | 공통국어, 공통수학, 통합사회, 통합과학 | 5등급 상대평가 |
| **일반선택** | 교양/기본 심화 | 문학, 대수, 미적분I, 한국지리, 물리학 | 5등급 상대평가 |
| **진로선택** | 전공 연계 심화 | 미적분II, 기하, 물리학II, 화학II, 생명과학II | 5등급 상대평가 |
| **융합선택** *(신설)* | 교과 간 융합 | 여행지리, 기후변화와 지속가능한 세계, 과학의 역사와 문화 | **성취평가제(A~E)** |

핵심 변화: 기존 9등급 → **5등급 상대평가** 전환 (융합선택만 절대평가)

## 2. 고교학점제 전면 시행 (2025년~)

| 항목 | 변화 내용 |
|------|-----------|
| **단위 → 학점** | 수업량을 "단위" 대신 "학점"으로 계산 |
| **졸업 기준** | 192학점 이수 (수업 174학점 + 창체 18학점) |
| **과목 선택권 확대** | 학생이 자신의 진로에 맞춰 과목 직접 선택 |
| **성취 기준** | 과목별 성취율 40% 이상 달성 시 학점 인정 |
| **이수 미인정** | 40% 미만 시 보충과정 이수 후 재평가 |

## 3. 2028 수능 과목 체계 변화 (통합형)

**가장 큰 변화: 선택과목 전면 폐지**

| 영역 | 현행 (2025~2027) | 2028 변경 |
|------|------------------|-----------|
| **국어** | 화법과작문/언어와매체 선택 | 선택 폐지, 통합 출제 (45문항 80분) |
| **수학** | 확통/미적분/기하 선택 | 선택 폐지, 통합 출제 (30문항 100분) |
| **탐구** | 사탐/과탐 17개 중 2과목 선택 | **통합사회 + 통합과학** 2과목 필수 응시 |
| **탐구 시간** | 과목당 30분, 20문항 | 과목당 **40분, 25문항** |
| **영어/한국사** | 절대평가 유지 | 동일 |
| **직업탐구** | 2과목 선택 | "성공적인 직업생활" 1과목 (25문항 40분) |
| **제2외국어/한문** | 30문항 40분 | **20문항 30분** |
| **시행일** | - | **2027년 11월 18일** |

## 4. 대학별 전공연계 핵심권장과목/권장과목 제도

대교협이 adiga.kr을 통해 **49개 대학**의 권장과목 공개:

| 구분 | 의미 | 중요도 |
|------|------|--------|
| **핵심권장과목** | 해당 전공에 반드시 이수를 권장하는 과목 | 미이수 시 정성평가에서 **불이익** |
| **권장과목** | 이수하면 유리한 과목 | 이수 시 **가점 요소** |

법적 강제성은 없으나, 서울대는 "권장과목 이수 여부가 수시 서류평가 및 정시 교과역량평가에 반영된다"고 **공식 명시**

### 주요 계열별 핵심권장 예시

| 계열 | 핵심권장과목 |
|------|-------------|
| 공학계열 | 미적분II, 물리학, 화학 |
| 의학계열 | 생명과학, 화학, 미적분II |
| 수학/통계 | 미적분, 확률과 통계, 기하 |
| 인문계열 | 제2외국어/한문 1과목 이상 |

## 5. 학생부종합전형 — 교과선택 평가 방식

2028부터 학종 평가가 **교과 이수 내역 중심**으로 대폭 강화:

| 평가 요소 | 내용 |
|-----------|------|
| **교과 성취도** | 과목별 등급/성취도 (5등급제) |
| **과목 선택의 적합성** | 지원 전공과 연관된 과목을 선택했는지 |
| **학업 수행 깊이** | 진로선택·심화 과목 이수 여부 |
| **이수 과목 다양성** | 전공 관련 과목의 폭과 깊이 |

핵심: "성적이 몇 등급인가"보다 **"무엇을 이수했는가"**가 더 중요해짐

## 6. 정시 교과평가 — 권장과목 반영

2028 정시에서 **학생부 정성평가가 확대**되는 대전환:

| 대학 | 정시 교과평가 | 변화 |
|------|-------------|------|
| **서울대** | 수능 60% + 교과평가 **40%** | 기존 20% → **40%로 2배 확대** |
| **경희대** | 정시에도 학생부 평가 도입 예고 | 신규 |
| **건국대** | 정시 학생부 반영 예고 | 신규 |

정시도 이제 **수능 점수만으로는 부족** — 교과 이수 내역이 합격을 좌우
