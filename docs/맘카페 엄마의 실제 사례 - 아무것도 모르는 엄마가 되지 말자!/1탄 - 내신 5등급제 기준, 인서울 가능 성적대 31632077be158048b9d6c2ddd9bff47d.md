# 1탄 - 내신 5등급제 기준, 인서울 가능 성적대

5등급제는 ‘평균 등급’의 밀도가 매우 높아집니다.

기존 9등급제 1.0~1.8이 5등급제 1.0~1.3으로 압축되었다고 보시면 되요. 고로 1.5부터는 이미 상위권에서 이탈된 것이고요.

여러 등급 데이터로 보면 인서울은 사실상 2.0 이전에서 대부분 갈린다고 보시면 됩니다.

이제는 **소수점으로** 대학을 써야하는 시대입니다.

---

제가 정리한 자료는~

전제1 : **교과전형**, **면접 없음 또는 영향 미미**, **수능최저 충족** 기준입니다.

전제2 : 고2 고3 성적이 반영되면 **소수점 자리가 하향조정** 될 수 있습니다.

전제3 : **일반고** 기준입니다. 특목고와 자사고는 합격 내신 등급이 다릅니다.

---

**인서울 가능 성적대 정리 (5등급제 기준)**

1. 인서울 상위권 **5등급제 평균 1.0 ~ 1.2**

(가능 대학은 교과 70%컷 기준으로 추정한 자료이므로 밑이 더 내려갈 수 있습니다.)

- 서울대학교의 경우 교과가 없습니다. 학종이에요.

연세대학교 : 1.1~1.2 내외

고려대학교 : 1.04~1.16

서강대학교 : 1.1~1.2

성균관대학교 : 1.13~1.2

한양대학교 : 1.1~1.2

서울교육대학교 : 1.16

전과목 거의 1등급으로 비교과 있어도 내신이 절대적입니다. 수능최저 충족 필수입니다.

---

2. 인서울 중상위권 **5등급제 평균 1.1 ~ 1.6**

중앙대학교 : 1.1~1.6

이화여자대학교 : 1.1~1.5

경희대학교 : 1.16~1.55

서울시립대학교 : 1.3~1.6

서울과학기술대학교 : 1.4~1.7

1.5 넘어가면 학과 선택 매우 중요해집니다.. 비교과 + 수능최저 여부가 당락 좌우합니다.

---

3. 인서울 하위권 / 커트라인 구간 **5등급제 평균 1.6 ~ 1.8**

동국대학교 : 1.23~1.7

건국대학교 : 1.16~1.43 (학과별 편차 큼)

숭실대학교 : 1.6~2.0

광운대학교 : 1.8~2.2

상명대학교 : 1.9~2.3

학과·모집인원 수능최저에 따라 크게 흔들립니다. 튿히 이 구간부터 수도권/지거국과 겹치니 어느 대학을 쓸지에 따라 유불리가 달라집니다.

---

결론으로 말하자면,

현재 5등급제에서 인서울 안정권은

1.0~1.3 : 상위 인서울

1.3~1.6 : 중상위 인서울

1.6~1.8 : 하위 인서울 / 학과·전형 따라 갈림

1.8 초과 : 인서울 ‘안정권’ 아님

사실상 1.8 아래는 인서울보다 지거국·수도권 상위가 현실적이지요.

또한,

인하대는 1.3~1.7라인

인천대는 1.6~ 2.초라인

으로 생각하시고 아이 내신성적을 관리하시면 될 것 같습니다.

- **물론 학생부 잘 채워서 자신의 내신 등급보다 높은 대학 갈 수 있습니다**. 학종이나 논술등 전형도 다양하고요. **하지만 정량도 제대로 못 챙기는 학생이 정성 부분을 제대로 챙기기란 어렵습니다.**

- **공부는 아이 스스로가 하는 거지만, 관리는 엄마의 역할~ 모두 화이팅입니다~~** ☺️
