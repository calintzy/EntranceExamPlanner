# 18탄 — 2027년 입시, 현 고3은 벼랑 끝 세대입니다.

2028년 입시변동 자료를 정리해 카페에 글을 올리면서, 2027학년도 대입을 보는 현 고3 아이들이 안타깝게 느껴졌습니다. 지역의사제 도입으로 의대 정원이 늘면서 N수생이 더 늘어날 것으로 전망되는데다, 2027학년도 대입이 현행 **‘선택형 수능 체제’의 마지막 시험**이기 때문입니다.

**무엇보다 2028학년도부터 입시 제도가 완전히 변경되기 때문에, 2027 이후 재수는 불리함이 매우 커질 수 있습니다**.

사실 과거 사례를 보면 수험 스타일이 바뀔 때마다 제도 자체가 재수생에게 유리하게 작용했습니다.

2015 개정 교육과정이 처음 적용됐을 때 재수생 비율은 27%로 최고치를 기록했고, 통합형 수능이 시행된 2022학년도에는 29.2%로 다시 최고치를 경신했습니다. 재수생들은 기본과목에서 핵심역량을 이미 갖추었기 때문에 제도변화의 단점을 금방 극복하는 편입니다.

그러나 **2028학년도부터는 재수생이 강점을 보이던 정시 비율이 축소**되고, 수능이 공통과목 체제(선택과목 폐지)로 전환됩니다. 즉, **2027학년도에 재수할 경우 완전히 다른 내용으로 다시 공부해야 하므로 이전보다 재수생이 겪는 부담이 훨씬 커질 것**으로 보입니다.

---

**2028학년도 대입제도 핵심 변화 요약**

① 통합사회·통합과학 필수 응시

2028 수능부터 모든 수험생은 통합사회와 통합과학을 필수로 응시해야 합니다. 문 이과 구분이 사실상 사라지며, 탐구 영역의 선택권도 없어졌습니다.

이과 학생 → 사회 분야(경제·정치·법 등) 추가 학습 필요

문과 학생 → 과학 기초(물리·화학·생명·지학 융합) 학습 필요

② 수학 대변혁 — 선택과목 폐지, 공통 범위 확대

‘미적분·기하·확률과 통계’ 선택과목이 사라집니다.

2028학년도부터는 대수Ⅰ, 적분Ⅰ, 확률과 통계가 전 수험생 공통 과목입니다. 자연계 학생이 누리던 ‘미적분 선택에 따른 유리함(표준점수 효과)’ 구조가 사라지고, 수학 전 범위가 강화되면서 문 이과 모두 동일한 범위에서 경쟁하게 됩니다.
