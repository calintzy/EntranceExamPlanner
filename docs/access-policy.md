# 입시연구소 — 로그인/비로그인 접근 정책

> 최종 업데이트: 2026-03-07
> Agent Council (Codex + Gemini) 합의 기반

---

## 설계 원칙

1. **핵심 가치는 개방**: 교과 가이드 + 역방향 검색은 비로그인으로 100% 사용 가능
2. **심화 분석은 맛보기**: 결과 요약을 보여주고 "더 보려면 로그인" 유도
3. **SEO 페이지 완전 공개**: 대학별/과목별 상세 페이지는 제한 없음
4. **차단보다 확장 프레이밍**: "로그인하면 3개 대학까지 확장할 수 있어요" (긍정적 문구)
5. **이탈 최소화**: 비로그인도 가치를 체험할 수 있어야 전환율이 높아짐

## 기능별 접근 정책

### Tier 1: 완전 개방 (Open Core)

| 기능 | 경로 | 비로그인 | 로그인 추가 혜택 |
|------|------|---------|-----------------|
| 교과 가이드 | `/guide` | 무제한 조회/비교 | (향후) 비교셋 저장, 즐겨찾기 |
| 역방향 검색 | `/search` | 무제한 검색 | (향후) 검색 조건 저장 |
| 대학별 상세 | `/university/[name]` | 전체 공개 | (향후) 관심 대학 표시 |
| 대학-학과 상세 | `/university/[name]/[dept]` | 전체 공개 | - |
| 과목별 상세 | `/subject/[name]` | 전체 공개 | (향후) 연관 대학 추적 |
| 수록대학 | `/universities` | 전체 공개 | - |
| 2028 정책 | `/policy` | 전체 공개 | - |
| 계약학과 | `/contract` | 전체 공개 | - |

### Tier 2: 부분 공개 (Freemium)

| 기능 | 경로 | 비로그인 | 로그인 |
|------|------|---------|--------|
| **맞춤 전략** | `/my-strategy` | 대학 **1개** 분석, 적합도 점수 + 강점 TOP3 표시, 세부 지표 잠금 | 대학 **3개** 동시 비교, 전체 세부 지표, (향후) 전략 저장 |
| **시너지 맵** | `/synergy` | 시너지 과목 **3개** 표시, 클러스터/TOP10 공개 | 시너지 과목 **10개** 표시, 전체 네트워크 |
| **이수 타임라인** | `/timeline` | **고1만** 표시, 고2~3 잠금 | 고1~고3 전체, (향후) 타임라인 저장 |
| **교체 시뮬레이션** | `/my-strategy` 내 | 교체 **1회**, 총점 변화 요약만 표시 | 무제한 교체, 상세 변화 (득실 학과 전체) |
| **친구와 비교** | `/compare` | 비교 링크 **조회만** 가능 (생성 불가) | 링크 생성 무제한, 전체 비교 결과 |

### Tier 3: 로그인 필수 (Login Required with Demo)

| 기능 | 경로 | 비로그인 | 로그인 |
|------|------|---------|--------|
| **전략 포트폴리오** | `/portfolio` | 기능 소개 + 샘플 데모 체험 (상위 1개 조합 미리보기) | 전체 기능 (대학 10개 조합, 최적 과목 추천) |

## 로그인 유도 트리거 시점

| 기능 | 트리거 시점 | 유도 문구 |
|------|------------|----------|
| 맞춤 전략 | 1개 분석 완료 후 2번째 대학 선택 시 | "로그인하면 3개 대학까지 동시에 비교할 수 있어요" |
| 시너지 맵 | 시너지 3개 표시 후 | "로그인하면 N개 더 볼 수 있어요" |
| 타임라인 | 고1 결과 확인 후 고2 탭 클릭 시 | "로그인하면 고2~고3 타임라인도 확인할 수 있어요" |
| 포트폴리오 | 페이지 진입 시 (데모 후) | "내 과목으로 맞춤 포트폴리오를 만들어보세요" |
| 교체 시뮬 | 1회 교체 후 2번째 시도 시 | "로그인하면 다양한 교체 시나리오를 비교할 수 있어요" |
| 친구와 비교 | 비교 링크 생성 버튼 클릭 시 | "로그인하면 친구에게 공유할 비교 링크를 만들 수 있어요" |

## 전환 카피 정책

- **금지**: "이 기능은 로그인이 필요합니다" (차단형)
- **권장**: "로그인하면 ~할 수 있어요" (확장형/성과형)
- **금지**: "무료 회원가입" (가격 프레이밍)
- **권장**: "카카오 · 구글 간편 로그인" (편의성 강조)

## 현재 구현 상태 vs 정책 비교

| 기능 | 정책 | 현재 구현 | 변경 필요 |
|------|------|----------|----------|
| 교과 가이드 | 완전 개방 | 완전 개방 | - |
| 역방향 검색 | 완전 개방 | 완전 개방 | - |
| 맞춤 전략 | 1개/3개, 요약/상세 | 1개/3개, 부분 잠금 | 세부 지표 잠금 강화 |
| 포트폴리오 | 샘플 데모 + 로그인 | 로그인 필수 전면 차단 | ⚠️ 샘플 데모 추가 필요 |
| 시너지 맵 | 3개/10개 | 3개/10개 | - |
| 타임라인 | 고1/고1-3 | 고1/고1-3 | - |
| 친구와 비교 | 조회만/생성 가능 | 제한 없음 | ⚠️ 비로그인 링크 생성 제한 |
| 교체 시뮬 | 1회 요약/무제한 상세 | 3개 제한 | ⚠️ 1회 제한 + 요약만 |
| SEO 페이지 | 완전 개방 | 완전 개방 | - |

### 우선순위 변경 항목

1. **포트폴리오**: 전면 차단 → 샘플 데모 체험 추가 (이탈 방지)
2. **친구와 비교**: 비로그인 시 링크 생성 제한 추가
3. **교체 시뮬**: 비로그인 1회 제한 + 결과 요약으로 축소
