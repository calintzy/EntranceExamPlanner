## 2. 모바일 전략 전문가: "PWA가 정답"

### 5가지 접근법 비교

| 접근법 | 코드 재사용 | 출시 시간 | 비용 | 팀 적합도 | 판정 |
|--------|-----------|----------|------|----------|------|
| **A. PWA** | **100%** | **1-2주** | **$0** | **최고** | **추천** |
| B. React Native | 0% | 6-10주 | $124+/년 | 중간 | 시기상조 |
| C. Flutter | 0% | 8-14주 | $124+/년 | 낮음 | 부적합 |
| D. 네이티브 | 0% | 12-20주 | $124+/년 | 매우 낮음 | 부적합 |
| E. Capacitor | 90%+ | 3-5주 | $124+/년 | 높음 | Phase 4 후보 |

### PWA 선택 근거

1. **Next.js 16이 PWA를 네이티브 지원** — `app/manifest.ts` + Service Worker만 추가
2. **데이터 특성이 PWA에 최적** — 연 1회 업데이트 정적 데이터(~250KB), 오프라인 동작 가능
3. **SEO 유지** — PWA는 웹 URL 그대로 유지, 네이티브 앱은 SEO 불가
4. **4월 시즌 대응 가능** — 1-2주면 출시
5. **확장 경로** — 검증 후 Capacitor로 앱스토어 래핑 가능

### 플랫폼별 기능 배분

| 기능 | 웹 (SSG + PWA) | 네이티브 앱 (Phase 4+) |
|------|---------------|----------------------|
| 대학별 권장과목 조회 | O (핵심) | Phase 4 이후 |
| 대학간 비교 | O (핵심) | Phase 4 이후 |
| 역방향 검색 | O (핵심) | Phase 4 이후 |
| SEO 랜딩 페이지 | O (웹 전용) | 불가 |
| 맘카페/카카오톡 공유 | O (OG 이미지) | 불가 |
| 내 아이 과목 체크리스트 | O (localStorage) | 향후 동기화 |
| 오프라인 열람 | PWA 캐시 | Phase 4 이후 |
| 푸시 알림 | PWA 푸시 | Phase 4 이후 |
| 홈 화면 설치 | PWA 매니페스트 | 앱스토어 |

### 네이티브 앱 전환 조건 (전부 충족해야 함)

- 월간 UV 10,000+
- 모바일 트래픽 비율 60%+
- 사용자 설문 "앱 원함" 30%+
- 투입 인력 3명+ 또는 외주 예산 확보

**현재 5개 조건 중 0개 충족 → 네이티브 앱은 논의 대상 아님**

## 3. 비즈니스 모델 전문가: "건당 결제 + 어필리에이트 하이브리드"

### 구독 vs 건당 결제: 왜 건당 결제인가

| 기준 | 구독 모델 | 건당 결제 | 판정 |
|------|----------|----------|------|
| 데이터 업데이트 빈도 | 연 1회 업데이트에 월 구독 정당화 어려움 | 필요할 때 1회 결제 = 자연스러움 | **건당** |
| 시즌성 | 4-7월만 쓰는데 12개월 구독? | 시즌에 집중 결제 가능 | **건당** |
| 사용 패턴 | 매년 새 코호트 → 이탈 후 새 유입 (구독 아닌 패턴) | 코호트별 1회 결제 = 적합 | **건당** |
| 무료 대안 | adiga.kr 무료, 구독 전환 동기 약함 | "추가 가치"에 대한 1회 결제 = 허들 낮음 | **건당** |
| 구현 복잡도 | 인증 + 과금 관리 + 해지 처리 필요 | 토스페이먼츠 1회 결제 = 단순 | **건당** |

### 추천 비즈니스 모델: 3단계 하이브리드

```
[1단계: 무료 — 트래픽 엔진] (Phase 1-2)
├── 전체 대학 권장과목 조회
├── 대학간 비교 (무제한)
├── 역방향 검색 (기본)
├── 정적 SEO 페이지 열람
└── → SEO 유입 + 맘카페 바이럴의 전제조건

[2단계: 건당 결제 — 첫 수익] (Phase 3 이후)
├── 맞춤 분석 리포트 PDF: 3,900~9,900원/건
│   (내 선택 과목 vs 목표 대학 상세 분석)
├── 교재/강의 어필리에이트: 마진 3-10%
└── → 결제 인프라 최소화 (토스페이먼츠 1회 결제)

[3단계: B2B — 스케일] (Phase 4, 조건부)
├── 학교/학원 대시보드: 연 50-200만원/기관
├── 교육청 단체 라이선스
└── → MAU 5,000+ 이후에만 검토
```

| **웹+모바일 동시** | Phase 1에서 PWA로 동시 대응 가능 | 웹도 배포 안 됐는데 모바일은 시기상조 |

### 종합 판정: 비즈니스 모델 추천

**구독보다 건당 결제를 추천합니다.** 근거:

1. **시즌성 서비스에 구독은 구조적으로 부적합** — 4-7월만 쓰는 서비스에 12개월 구독은 부자연스럽고 이탈률이 70-85%로 예상
2. **데이터 업데이트가 연 1회** — 월 구독 가치를 매월 정당화할 콘텐츠가 부족
3. **무료 대안(adiga.kr) 존재** — 구독 전환 동기가 약함. 건당 결제가 허들 낮음
4. **구현 복잡도** — 구독은 인증+과금관리+해지 처리 필요, 건당은 토스페이먼츠 1회 연동으로 충분
5. **매년 새 코호트 유입** — 이탈 후 재구독이 아닌 새 사용자 유입 패턴 = 건당 결제에 적합

**단, 수익화는 Phase 2 이후의 과제. 지금은 배포가 먼저.**
